# Ecommerce-Sales
Ecommerce Sales Dashboard

An interactive dashboard built using Power BI to analyze ecommerce sales performance.

 *Key Features
-Total Sales, Quantity & AOV KPIs
-Sales by State & Customer
-Profit by Month & Sub-Category
-Quantity by Category & Payment Mode
-Quarter & State filters for dynamic analysis

* Tools Used
-Power BI
-DAX
-Excel / CSV
